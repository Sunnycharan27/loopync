## üéØ Executive Summary

**Loopync** is India's premier student-focused social media platform designed to bridge the gap between academic life and professional development. The platform combines Instagram-like social features with LinkedIn-style professional networking, specifically tailored for the Indian student demographic.

### Key Differentiators
- **Category-Based Tribes**: Dynamic community features based on interests (Tech, Fitness, Food, Business, College, Creative, Startup)
- **Instagram-like Stories (VibeCapsules)**: With full music integration via Deezer API
- **Skill-Based Discovery**: Connect with students based on skills and certifications
- **Startup Ecosystem**: Built-in showcase and ideation platform for student entrepreneurs

## üîß How to Use Loopync

### Getting Started

#### Step 1: Create Account
1. Open Loopync app
2. Click "Sign Up"
3. Enter email, password, handle
4. Complete profile with bio, avatar, skills
5. Choose interests to personalize feed

#### Step 2: Join Tribes
1. Navigate to "Tribes" tab
2. Browse categories (Tech, Fitness, Food, etc.)
3. Click "Join" on interesting tribes
4. Start participating in discussions

#### Step 3: Create Content
- **Posts**: Share updates, photos, videos
- **Stories**: Add photos with music
- **Projects**: Showcase your work
- **Resources**: Share learning materials

### For Students

#### Building Your Profile
1. Add professional bio
2. Upload avatar and cover photo
3. Add skill tags (React, Python, Design, etc.)
4. List certifications with verification URLs
5. Showcase projects with GitHub links

#### Finding Opportunities
1. Join "College" or "Tech" tribes
2. Check "Jobs" tab for internships
3. Apply directly through the platform
4. Network with recruiters in comments

#### Forming Teams
1. Go to "Teams" tab in your tribe
2. Click "Find Team" to post requirement
3. Specify roles needed (Developer, Designer, etc.)
4. Review applications and accept members

### For Creators

#### Showcasing Work
1. Join "Creative" or "Business" tribes
2. Go to "Portfolio" tab
3. Upload multiple images of your work
4. Add tools used (Figma, Photoshop, etc.)
5. Link to project URLs

#### Finding Collaborations
1. Go to "Collabs" tab
2. Browse or post collaboration requests
3. Specify role requirements
4. Connect with interested creators

### For Entrepreneurs

#### Showcasing Startup
1. Join "Startup" or "Tech" tribes
2. Go to "Startups" tab
3. Create showcase with:
   - Company name and description
   - Stage (Idea/MVP/Growth/Scale)
   - Key metrics
   - Funding stage
   - Team size
   - Achievements

#### Sharing Ideas
1. Go to "Ideas" tab
2. Post your idea with:
   - Title and description
   - Problem you're solving
   - Your solution
   - What you're looking for
3. Get votes and feedback from community

## üì± Technical Architecture

### Frontend
- **Framework**: React 18 with Create React App
- **Styling**: Tailwind CSS with custom components
- **State Management**: React Context + Local State
- **Routing**: React Router v6
- **UI Components**: Shadcn/UI + Custom components
- **Icons**: Lucide React

### Backend
- **Framework**: FastAPI (Python)
- **Database**: MongoDB with Motor (async driver)
- **Authentication**: JWT with bcrypt hashing
- **Real-time**: Socket.IO
- **File Storage**: Local uploads directory
- **Rate Limiting**: SlowAPI

### Integrations
- **Music**: Deezer API (free 30-sec previews)
- **Audio Rooms**: Agora.io SDK
- **Payments**: Razorpay (ready for integration)
- **AI**: Emergent Integrations LLM

### Database Collections
```
users, posts, reels, tribes, vibecapsules, projects, certifications,
team_posts, internships, workouts, challenges, menu_items, deals,
reviews, trainers, events, services, portfolios, ideas, showcases,
resources, collaborations, follow_requests, endorsements, reputation,
notifications, messages, dm_threads
```
